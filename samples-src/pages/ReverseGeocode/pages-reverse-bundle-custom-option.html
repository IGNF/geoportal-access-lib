Paramétrage du Service de geocodage inverse

Indexes et filtres:

- CadastralParcel
   * identifiant
   * codeDepartement
   * codeCommune
   * nomCommune
   * codeCommuneAbs
   * codeArrondissement
   * section
   * numero
   * feuille
- PositionOfInterest
   * type
   * postalCode
   * inseeCode
- StreetAddress
   * city
   * postalCode
   * inseeCode

{

    },
    "maximumResponses" : 2,
    "returnTrueGeometry" : true,
    "searchGeometry" : {
      "type" : "Circle",
      "coordinates" : [
        2.35,
        48.87
      ],
      "radius" : 100
    },

      "lat" : 48.87
    }
}

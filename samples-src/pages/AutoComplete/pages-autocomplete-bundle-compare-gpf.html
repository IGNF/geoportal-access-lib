Utilisation du Service d'autocompletion

Avec les paramètres obligatoires seulement

          
              var options = {
                apiKey : "calcul",
                onSuccess : function (response) {
                    print(response);
                },
                // spécifique au service
                text : "Saint-Man"
            };
            Gp.Services.autoComplete(options);
